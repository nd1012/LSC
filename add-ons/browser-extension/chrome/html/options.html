LSC - JavaScript localStorage Cache options

Pre-fetching means LSC will try to find all linked contents and load their HTML from the server before you even clicked the link. This results in a very fast browsing experience, but may produce some overhead, if contents from links that you never visit is being loaded.
Using the session storage for cached contents will avoid storing sensitive information from a website. The session storage cache will be deleted as soon as you close the window (or tab) - it's a kind of private mode for the LSC cache.
